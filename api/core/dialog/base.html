class Dialog.Base

Constructor

new Dialog.Base(options)
  • options (Object) – options for this dialog

Supported options:

  • width (String | Number): the desired width of the dialog. By default set to Dialog.Options.defaultWidth.
  • modal (Boolean): true to simulate a modal dialog, false otherwise (the default).
  • transitionDuration (Number): if using effects will determine the duration of each effects transition. By default set to Dialog.Options.transitionDuration.
  • buttons (Array): list of button descriptions for the dialog. See Dialog.ButtonPanel#addButton for details. By default a dialog includes one button with the text “Close” which closes the dialog.
