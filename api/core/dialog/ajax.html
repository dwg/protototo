    class Dialog.Ajax

    Description

    A base class for dialogs supporting ajax content.

    Superclass

    Dialog.Base

    Methods

    • new

    Constructor

    new Dialog.Ajax(path, options)
    • path (String) – the ajax request path
    • options (Object) – options for this dialog

    Options in addition to ones described for Dialog.Base:

    • ajax (Object): options to pass to the ajax request. By default set to {method: 'get'}.
    • closeOnFailure (Boolean): true if the dialog should be closed on ajax failure (the default).
